# PHPMyAdmin Upgrader

## Introduction

- It's the Bash script for upgrading PHPMyAdmin package on Ubuntu 16.04 or 18.04 manually.

## Usage

```Bash
./upgrader.sh blow_secret_token
```

## Current requirements

- PHP `7.2`, `7.3` or `7.4` version.
- PHPMyAdmin `4.9.7` verison.
